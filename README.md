# Лендинг — Татьяна Ищенко

Премиальный одностраничный сайт в стиле [Aniko Yoga](https://anikoyoga.ru): тёмная палитра, журнальная типографика, адаптивная вёрстка.

## Содержимое

- `index.html` — главная страница
- `images/profile.png` — фото с [Taplink](https://taplink.cc/tatianaishchenko)

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub (например `tatiana-yoga`).
2. Загрузите папку целиком (или только `index.html` + `images/`).
3. **Settings → Pages → Build and deployment → Source:** `Deploy from a branch`.
4. Branch: `main`, folder: `/ (root)`.
5. Сайт будет доступен по адресу `https://<username>.github.io/tatiana-yoga/`.

Для кастомного домена добавьте файл `CNAME` с именем домена.
